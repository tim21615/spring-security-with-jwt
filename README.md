# Shopping Center
本來在練習 Spring Security 結合 JWT 的登入認證，做完後，突然覺得乾脆做一個 電商網站的 Side Project 好了...因此有了此專案!
#### 目前有的功能
1. Spring Security 結合 JWT 做登入認證
2. 使用者資訊的 Api
3. 商城商品的各項顯示用 Api
4. 購物車相關 Api

#### 使用技術
1. Spring Boot
2. Spring Data JPA
3. Redis 緩存
4. Spring Security
5. JWT

#### 未來要開發的功能
1. 結帳 (注意交易控制)
2. 後台新增商品、修改商品、商品上下架等功能
3. 查看訂單
